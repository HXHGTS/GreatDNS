## Debian安装源-阿里云

### Debian 7

```
rm -f /etc/apt/*
curl -sSL https://raw.githubusercontent.com/HXHGTS/GreatDNS/master/source/debian/aliyun/debian7_aliyun_sources.list > /etc/apt/sources.list && apt update
```
